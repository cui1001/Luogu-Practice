# 洛谷刷题记录 (Luogu-Practice)

📝 这里是我在洛谷平台（Luogu）上的算法题解仓库，主要记录学习过程、复盘算法思路。

## 📂 仓库结构
该仓库按题目来源或算法分类进行管理：
- `dp/` : 动态规划（Dynamic Programming）题解
- `graph/` : 图论相关题解
- `search/` : 搜索与回溯题解
- `basic/` : 入门基础语法与模拟题

## 💻 语言与环境
- **主语言**: C++
- **开发环境**: VS Code 
- **代码规范**: 注释清晰，模块化书写

## 🚀 核心目标
1. **算法入门**：从基础语法到数据结构，逐步攻克难点。
2. **竞赛备赛**：为蓝桥杯、ACM/ICPC 等竞赛积累题库与题解经验。
3. **复盘总结**：每道题记录解题思路、复杂度分析与优化方向。

## 📈 进度概览
| 算法分类 | 完成进度 | 主要题型 |
| :--- | :--- | :--- |
| 动态规划 | 🟡 进行中 | 背包问题、线性DP、区间DP |
| 搜索与图论 | ⚪ 未开始 | DFS、BFS、最短路、最小生成树 |
